Ruby Enumerable Method Cheat Sheet

Ruby is an object-oriented programming language for computers. As part of its makeup, a certain amount of functions have already been defined for the user, and can be called with a simple mention of their name. Here below is a list of all such "enumerable" methods, which are functions that specifically take a set of data and does something to the data.

To see the official Ruby Docs write-up for any particular method, click the method's name.

#all?
